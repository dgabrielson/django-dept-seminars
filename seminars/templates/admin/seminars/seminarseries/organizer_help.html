{% endblock %} The organizing group for a seminar series is a regular user group.
Members of the organizing group can use the admin site to organize seminars.
Depending on your needs, one group can look after more than a single seminar series.

Setting up an Organizing Group

  • Create the group (if necessary) using Home › Authentication and Authorization › Groups › Add group.
    If you are using an existing user group, select it from Home › Authentication and Authorization › Groups.
    Note: From the seminar series edit page, you can click the green plus or the pencil icons beside the organizing group drop-down.
  • Make sure the group has a meaningful name, like “Seminar organizers: [series name]” (without quotes).
  • Ensure that the group has the correct permissions.
    Type “seminars” in the available permissions search box to filter the permissions list down to just the seminars app.
    Select all of the permissions, except add and delete series, and click the left arrow to make these chosen permissions.
    (On Windows, press CTRL+A to select all of the seminars permissions, then CTRL+click “Can add series” and “Can delete series”.
     On a Mac, use the CMD key instead of CTRL.)
    The correct permissions are shown below.
  • Make sure to SAVE or ADD the group.

Group Membership

  • Membership of groups is handled by modifying individual users.
    Go to Home › Authentication and Authorization › Users.
    If you have only a few users, you can select a name from the list, otherwise, you may need to search for a particular user.
    (Searches can be done by username, email, first name or last name.)
  • Click the username to open the Change user page.
  • In the “permissions” section, ensure that “Staff status” is checked.
    (Only staff users can access the admin site.)
  • In the “permissions” section, under “Available groups”, select the appropriate organizing group and move it to one of the user's “Chosen groups”.
    (Use the filter if necessary.)
  • SAVE the changes to the user.
